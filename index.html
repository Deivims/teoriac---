Definiciones de Conceptos

Programa

Conjunto de instrucciones escritas en un lenguaje de programación que realiza una tarea específica.

Tipos de programas

Clasificación de programas según su uso, como software de sistema, software de aplicación, y software de desarrollo.

Código fuente

El conjunto de instrucciones y declaraciones escritas por un programador en un lenguaje de programación, que se puede leer y modificar.

Código objeto

El resultado de la compilación del código fuente, en un formato que puede ser ejecutado por una máquina.

Software de sistema

Conjunto de programas diseñados para gestionar los recursos del hardware y proporcionar servicios a otros programas, como sistemas operativos.

Software de aplicación

Programas diseñados para realizar tareas específicas para el usuario, como procesadores de texto o navegadores web.

Palabras reservadas

Palabras que tienen un significado especial en un lenguaje de programación y no pueden ser utilizadas como identificadores.

Comentarios

Anotaciones en el código que son ignoradas por el compilador, utilizadas para documentar el código y hacer notas para otros programadores.

Concepto de librería

Conjunto de funciones y procedimientos predefinidos que se pueden utilizar en un programa para realizar tareas específicas, evitando la reescritura de código.

Concepto de Método o función

Bloque de código que realiza una tarea específica y puede ser reutilizado en diferentes partes de un programa. Los métodos suelen pertenecer a una clase.

Método principal main

El punto de entrada de un programa en muchos lenguajes de programación, donde comienza la ejecución del código.

Conceptos de Datos

Representaciones de información que pueden ser manipuladas por un programa, como números, texto, y estructuras complejas.

Variables

Espacios en memoria que almacenan datos que pueden cambiar durante la ejecución del programa.

Constantes

Espacios en memoria que almacenan datos que no cambian durante la ejecución del programa.

Tipos de datos

Clasificación de datos en un lenguaje de programación, que determina el tipo de valores que una variable puede almacenar.

Numéricos: o Entero

Tipo de dato que representa números enteros, es decir, sin parte decimal.
